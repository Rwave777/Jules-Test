# テトリスゲーム

これはブラウザで動作するシンプルなテトリスゲームです。

## 利用方法

1.  このリポジトリをクローンまたはダウンロードします。
2.  `index.html` ファイルをウェブブラウザで開きます。

## 操作説明

-   **← (左矢印キー)**: テトリミノを左に移動します。
-   **→ (右矢印キー)**: テトリミノを右に移動します。
-   **↓ (下矢印キー)**: テトリミノを下に早く落とします（ソフトドロップ）。
-   **↑ (上矢印キー)**: テトリミノを時計回りに回転させます。

## 開発について

-   **HTML**: `index.html` (ゲームの基本構造)
-   **CSS**: `style.css` (ゲームの見た目)
-   **JavaScript**: `script.js` (ゲームのロジック)

ゲームのログは、ブラウザの開発者コンソールで確認できます。
